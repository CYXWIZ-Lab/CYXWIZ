# CyxWiz B2B vs B2C Strategy

## Overview

CyxWiz operates as a **hybrid marketplace** serving both individual consumers (B2C) and businesses (B2B). This document outlines the differences in approach, strategy, and execution for each segment.

---

## Understanding B2B vs B2C

```
┌─────────────────────────────────────────────────────────────────┐
│                    B2B vs B2C FUNDAMENTALS                      │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   B2C (Business to Consumer)         B2B (Business to Business) │
│   ──────────────────────────         ────────────────────────── │
│                                                                 │
│   • Individual people                • Companies/organizations  │
│   • Personal money                   • Company budget           │
│   • Emotional decisions              • Rational/committee       │
│   • Fast sales (minutes)             • Slow sales (months)      │
│   • Self-serve                       • Sales team required      │
│   • Low touch                        • High touch               │
│   • Many small deals                 • Few large deals          │
│   • Price sensitive                  • Value sensitive          │
│   • Community support                • Dedicated support        │
│   • Viral growth                     • Relationship growth      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## Strategic Approach

### Phase 1: B2C-First (Year 1)

```
┌─────────────────────────────────────────────────────────────────┐
│                    PHASE 1: B2C-FIRST STRATEGY                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   WHY START B2C:                                                │
│   ──────────────                                                │
│   ✓ Faster feedback loops (try today, feedback tomorrow)        │
│   ✓ No sales team required (product-led growth)                 │
│   ✓ Network effects build faster (many small > few large)       │
│   ✓ Lower CAC (community marketing vs enterprise sales)         │
│   ✓ Validate product before enterprise complexity               │
│   ✓ Build case studies for future B2B sales                     │
│                                                                 │
│   B2C FOCUS AREAS:                                              │
│   ────────────────                                              │
│   • Indie ML developers                                         │
│   • Gamers with GPUs                                            │
│   • Students and researchers                                    │
│   • AI hobbyists and enthusiasts                                │
│                                                                 │
│   SUCCESS METRICS:                                              │
│   ────────────────                                              │
│   • 10,000+ registered users                                    │
│   • 1,000+ active node operators                                │
│   • $100K+ monthly transaction volume                           │
│   • NPS > 40                                                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Phase 2: B2B Layer (Year 2)

```
┌─────────────────────────────────────────────────────────────────┐
│                    PHASE 2: ADD B2B LAYER                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   WHEN TO ADD B2B:                                              │
│   ────────────────                                              │
│   • Product-market fit proven with B2C                          │
│   • Reliability metrics strong (>95% success rate)              │
│   • Organic B2B inbound increasing                              │
│   • Case studies from power users available                     │
│                                                                 │
│   B2B ADDITIONS:                                                │
│   ──────────────                                                │
│   • Enterprise tier with SLAs                                   │
│   • Dedicated account management                                │
│   • Custom pricing and contracts                                │
│   • Compliance features (SOC2 roadmap)                          │
│   • Priority support                                            │
│                                                                 │
│   SALES MOTION:                                                 │
│   ─────────────                                                 │
│   • Hire first AE (Account Executive)                           │
│   • Inbound-led initially                                       │
│   • Demo → Pilot → Contract flow                                │
│   • Target: AI startups first (faster sales cycle)              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

│                    PHASE 3: FULL HYBRID MODEL                   │

│                                                                 │

## Pricing Strategy

```
B2C PRICING MODEL
─────────────────

PRINCIPLE: Simple, transparent, self-serve

COMPUTE:
├── Pay-as-you-go (per GPU-second)
├── No minimums
├── No commitments
├── Public pricing page
└── Instant billing

TIERS:
├── Free:     $25 credits to start
├── Standard: Pay-as-you-go, 10% platform fee
└── Pro:      $99/month, 7% fee, priority queue

NODE OPERATORS:
├── 90% of job revenue
├── Instant settlement
├── No fees to join
└── Staking for priority (optional)
```

### B2B Pricing

```
B2B PRICING MODEL
─────────────────

PRINCIPLE: Value-based, customized, negotiated

COMPUTE:
├── Volume discounts (5-20% based on commitment)
├── Monthly invoicing (Net 30)
├── Annual contracts (10% discount)
├── Reserved capacity options
└── Custom pricing for large volumes

TIERS:
├── Startup:    $1,000/mo min, 8% fee, email support
├── Business:   $5,000/mo min, 6% fee, priority support
├── Enterprise: Custom pricing, 5% fee, dedicated CSM, SLA
└── Custom:     White-label, private networks, etc.

ENTERPRISE ADD-ONS:
├── 99.9% SLA:     +20%
├── Dedicated CSM: +$1,000/mo
├── SOC2 reports:  Included (Enterprise)
├── Custom contract: Legal review
└── Private network: Custom pricing
```

## Risk Considerations

### B2C Risks

| Risk | Impact | Mitigation |
|------|--------|------------|
| High churn | Revenue volatility | Improve activation, engagement |
| Low LTV | Hard to spend on acquisition | Upsell to Pro, marketplace |
| Price sensitivity | Race to bottom | Differentiate on UX, features |
| Support burden | Unscalable | Self-serve, community, docs |

### B2B Risks

| Risk | Impact | Mitigation |
|------|--------|------------|
| Long sales cycles | Slow growth | Start with startups |
| Enterprise demands | Feature creep | Roadmap discipline |
| Concentration | Single customer risk | Diversify, no >10% revenue |
| Compliance requirements | High cost | SOC2 roadmap, staged approach |
