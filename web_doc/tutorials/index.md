# Tutorials

Welcome to the CyxWiz tutorials! This section provides step-by-step guides to help you get started with the platform.

## Learning Path

Follow this recommended progression:

```
1. Getting Started     → Set up your first project
        ↓
2. Basic Node Tutorial → Build a simple neural network visually
        ↓
3. Basic Scripting     → Write Python scripts in CyxWiz
        ↓
4. Complete Workflow   → Train and evaluate a real model
```

## Prerequisites

Before starting, ensure you have:

- CyxWiz Engine installed and running
- Basic understanding of machine learning concepts (helpful but not required)
- Familiarity with Python syntax (for scripting tutorials)

## What You'll Learn

### Visual Model Building
- How to use the Node Editor to build neural networks
- Connecting layers with the visual interface
- Configuring layer parameters

### Python Scripting
- Using the embedded Python interpreter
- Writing `.cyx` notebook-style scripts
- Interacting with the `pycyxwiz` module

### Training Workflows
- Loading and preparing datasets
- Running local training sessions
- Monitoring training progress
- Evaluating model performance
