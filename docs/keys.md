# CyxWiz Engine - Keyboard Shortcuts Reference

This document lists all keyboard shortcuts and key combinations available in the CyxWiz Engine.

---

## Script Editor

### File Operations
| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | Create new script file |
| `Ctrl+O` | Open existing script file |
| `Ctrl+S` | Save current script |
| `Ctrl+Shift+S` | Save script as new file |
| `Ctrl+W` | Close current tab |

### Edit Operations (TextEditor)
| Shortcut | Action |
|----------|--------|
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Ctrl+X` | Cut selected text |
| `Ctrl+C` | Copy selected text |
| `Ctrl+V` | Paste from clipboard |
| `Ctrl+A` | Select all text |

### Script Execution
| Shortcut | Action |
|----------|--------|
| `F5` | Run entire script |
| `Shift+F5` | Stop running script |
| `F9` | Run selected code |
| `Ctrl+Enter` | Run current section |
| `F10` | Debug script |

---

## Asset Browser

### File/Asset Management
| Shortcut | Action |
|----------|--------|
| `F2` | Rename selected item |
| `Delete` | Delete selected assets |
| `Ctrl+C` | Copy selected asset |
| `Ctrl+X` | Cut selected asset |
| `Ctrl+V` | Paste asset from clipboard |
| `Ctrl+N` | Create new script file |
| `Ctrl+A` | Select all items |
| `F5` | Refresh directory view |
| `Escape` | Clear selection |

### Multi-Select
| Shortcut | Action |
|----------|--------|
| `Ctrl+Click` | Add/remove item from selection |
| `Shift+Click` | Select range of items |

---

## Node Editor

### Node Operations
| Shortcut | Action |
|----------|--------|
| `Delete` | Delete selected nodes |
| `Ctrl+Click` | Multi-select nodes |
| `Ctrl+Click` (on link) | Detach link from node |
| `Left-Click Drag` (empty space) | Pan the editor |

---

## Command Window

### Navigation
| Shortcut | Action |
|----------|--------|
| `Up Arrow` | Previous command in history |
| `Down Arrow` | Next command in history |
| `Tab` | Show/apply auto-completions |

## Quick Reference Card

### Most Common Shortcuts

| Shortcut | Context | Action |
|----------|---------|--------|
| `Ctrl+S` | Script Editor | Save |
| `Ctrl+N` | Script Editor / Asset Browser | New file |
| `F5` | Script Editor | Run script |
| `F5` | Asset Browser | Refresh |
| `Shift+F5` | Script Editor | Stop script |
| `F2` | Asset Browser | Rename |
| `Delete` | Asset Browser / Node Editor | Delete selection |
| `Ctrl+A` | Any | Select all |
| `Ctrl+C` | Any | Copy |
| `Ctrl+V` | Any | Paste |
| `Ctrl+X` | Any | Cut |
| `Ctrl+Z` | Script Editor | Undo |
| `Escape` | Asset Browser | Clear selection |

---

## Notes

- Keyboard shortcuts are context-sensitive and only work when the relevant panel is focused
- Some shortcuts may have different behaviors depending on the active panel
- Script execution shortcuts (`F5`, `F9`, `F10`) only work when no script is currently running
- `Shift+F5` only works when a script is running
