### Components

| Component | Technology | Purpose |
|-----------|------------|---------|
| **CyxWiz Website** | Next.js, shadcn/ui, NextAuth | Account hub, downloads, models browser |
| **Rust API** | Axum, MongoDB, JWT | Backend API for website and apps |
| **CyxWiz Engine** | C++, ImGui, OpenGL | Desktop IDE for ML model design |
| **CyxWiz Central Server** | Rust, gRPC, PostgreSQL | Network orchestrator for jobs |
| **CyxWiz Server Node** | C++, ArrayFire | Distributed compute worker |
| **cyxwiz-backend** | C++, ArrayFire | Shared ML compute library |
| **CyxWiz Blockchain** | Solana, Anchor | Payment escrow & node registry |
| **CyxCloud** | Rust, libp2p | Decentralized storage |
